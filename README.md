# 🐶🐱 Dogs vs Cats Classifier

## 📌 Project Overview

This project is a **Deep Learning-based image classifier** that predicts
whether an image is a **dog 🐶 or a cat 🐱**.\
It uses a Convolutional Neural Network (CNN) trained on labeled image
data.

------------------------------------------------------------------------

## 🚀 Features

-   Classifies images into Dog or Cat
-   Built using Convolutional Neural Networks (CNN)
-   Supports custom image prediction
-   Easy to train and test using Jupyter Notebook

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   Python
-   TensorFlow / Keras
-   NumPy
-   Matplotlib
-   OpenCV (optional)
